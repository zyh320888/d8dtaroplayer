# d8dtaroplayer

用于运行 https://d8dcloud.com 上开发的  微信小程序，支付宝小程序，app 等应用

```sh
# 首次运行
npm i

# 从d8dAiIDE上复制小程序代码
# 粘贴到 /src/appCode.js 覆盖

# 运行微信小程序
npm run dev:weapp 

# 运行支付宝小程序
npm run dev:alipay 
```
